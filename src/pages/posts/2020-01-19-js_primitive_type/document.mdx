# 기본 자료형 (Primitive Type)

JS는 7개의 기본 자료형을 소유하고있다.

1. Boolean
2. Null
3. Undefined: 값이 할당되지 않은 변수
4. Number
5. String
6. Symbol (ECMAScript 6 에 추가됨): 유일하고 변경 불가능한 기본값. 객체 속성의 key값으로도 사용 가능
7. Object

Object를 제외한 기본 자료형들은 모두 immutable 하다.
